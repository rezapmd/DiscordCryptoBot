# DiscordCryptoBot
Its a bot that gives the exact value(price) of a cryptocurrency , graphs the price with an image , kicks and bans a user if the bot has the permission to do so . 


commands = {!price , !graph , !kick , !ban}

usage for !price : 
!price eth
!price btc

usage for !graph : 
!graph btc
!graph eth
!graph shib

usage for kick : 
!kick @{user} {reason}

usage for ban : 
!ban @{user} {reason}
